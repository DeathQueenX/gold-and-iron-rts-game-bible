# 06 — Environment

## Scop

Organizarea tuturor elementelor care construiesc lumea jocului și câmpul de luptă.

## Categorii planificate

- `Terrain`
- `Biomes`
- `Vegetation`
- `Rocks_and_Cliffs`
- `Water_and_Coasts`
- `Resources`
- `Roads_and_Bridges`
- `Ruins`
- `Props`
- `Ambient_Details`
- `Weather_and_Atmosphere`

## Conținut

- reguli pentru teren modular;
- tile-uri și tranziții între biome;
- resurse colectabile;
- obiecte de decor;
- elemente destructibile;
- variații pentru evitarea repetiției;
- niveluri de detaliu;
- coliziuni și navigație;
- iluminare, ceață și vreme;
- specificații de optimizare pentru Unity.

## Prima țintă

Un mic set de teren, o resursă principală, câteva obiecte de vegetație și un element de ruină pentru vertical slice.

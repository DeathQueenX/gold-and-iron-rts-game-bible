# 08 — Visual Effects

## Scop

Organizarea efectelor vizuale folosite pentru combat, magie, selecție, interacțiuni și atmosferă.

## Categorii planificate

- `Melee_Impacts`
- `Projectiles`
- `Magic`
- `Healing_and_Buffs`
- `Debuffs_and_Control`
- `Selection_and_Commands`
- `Movement_and_Trails`
- `Damage_and_Status`
- `Death_and_Decay`
- `Building_Construction_and_Destruction`
- `Environment_and_Weather`

## Conținut per efect

- scop în gameplay;
- facțiune sau sursă;
- formă și paletă;
- durată și timing;
- intensitate;
- variante de nivel;
- legătura cu animația și sunetul;
- particule, mesh-uri și shader-e necesare;
- reguli de performanță;
- prefab și setări Unity.

## Regula principală

Efectele trebuie să comunice clar acțiunea și zona de pericol fără să ascundă unitățile sau să transforme luptele mari într-un amestec vizual.

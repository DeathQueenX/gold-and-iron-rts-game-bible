# 07 — UI and UX

## Scop

Definirea interfeței vizuale și a regulilor de prezentare a informației în timpul jocului.

## Categorii planificate

- `HUD`
- `Selected_Unit_Panel`
- `Command_Cards`
- `Minimap`
- `Health_Mana_and_Status_Bars`
- `Resource_Display`
- `Menus`
- `Tooltips`
- `Cursors_and_Command_Indicators`
- `Ability_and_Item_Icons`
- `Victory_Defeat_and_Loading_Screens`

## Principii

- informațiile esențiale trebuie citite rapid;
- interfața nu trebuie să acopere inutil câmpul de luptă;
- culorile facțiunilor și stările unităților trebuie să fie consecvente;
- iconițele trebuie să rămână lizibile la dimensiuni mici;
- elementele decorative trebuie să susțină tema jocului fără a reduce claritatea.

## Prima țintă

HUD minim pentru vertical slice: unitate selectată, viață, comenzi, resurse, minimap și feedback de selecție.

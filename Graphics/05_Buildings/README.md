# 05 — Buildings

## Scop

Definirea și organizarea tuturor clădirilor, fortificațiilor și structurilor interactive.

## Categorii planificate

- `Headquarters`
- `Military`
- `Economic`
- `Technology_and_Magic`
- `Defensive`
- `Walls_and_Gates`
- `Resource_Structures`
- `Neutral_and_NPC`
- `Ruins_and_Destroyed_Variants`
- `Modular_Architecture`

## Conținut per clădire

- rol și funcție;
- facțiune;
- footprint și scară;
- concept din unghi RTS;
- vedere frontală, laterală și de sus;
- module reutilizabile;
- materiale și texturi;
- variante de construcție și distrugere;
- animații și elemente mobile;
- puncte de producție și ieșire a unităților;
- iluminare și VFX;
- specificații Unity.

## Prima țintă

Clădirea principală și cazarma primei facțiuni, realizate modular pentru a stabili limbajul arhitectural.

# Graphics Vertical Slice Roadmap

## Scop

Validarea pipeline-ului grafic complet cu un număr foarte mic de asset-uri înainte de producția pe scară largă.

## Ordinea de lucru

### Milestone 1 — Primul prop

Asset: butoi medieval neutru.

Obiective:

- modelare simplă în Blender;
- UV de bază;
- material lemn și metal;
- export FBX;
- import în Unity;
- collider și prefab;
- verificare din camera RTS.

### Milestone 2 — Primul element natural

Asset: piatră sau copac simplu.

Obiective:

- siluetă lizibilă;
- variații simple;
- verificarea integrării cu terenul.

### Milestone 3 — Prima clădire

Asset: structură mică neutră sau depozit.

Obiective:

- scară arhitecturală;
- pivot pentru plasare;
- acoperiș lizibil din camera RTS;
- collider simplificat.

### Milestone 4 — Prima unitate

Asset: unitate melee de bază.

Obiective:

- concept sheet;
- model low-poly stilizat;
- rig humanoid;
- idle, walk, attack și death;
- import și Animator în Unity.

### Milestone 5 — Primul set de facțiune

- unitate melee;
- unitate ranged;
- clădire principală;
- cazarmă;
- banner;
- paletă și materiale comune.

## Regula de progres

Nu se începe următorul milestone până când asset-ul curent este vizibil și funcțional într-o scenă Unity.

## Definiția „Finished”

Un asset este finalizat când:

- arată corect din camera RTS;
- are scară și pivot corecte;
- folosește materiale conforme;
- este importat ca prefab;
- nu produce erori;
- poate fi reutilizat într-o hartă reală.

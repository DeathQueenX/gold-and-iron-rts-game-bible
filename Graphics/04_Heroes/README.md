# 04 — Heroes

## Scop

Centralizarea designului vizual pentru eroii principali și secundari.

## Conținut pentru fiecare erou

- identitate vizuală și personalitate;
- siluetă distinctivă;
- concept frontal, lateral și posterior;
- portret pentru UI;
- expresii faciale;
- armură pe fiecare zonă a corpului;
- arme și obiecte personale;
- variante de echipament;
- abilități și iconițe;
- VFX personale;
- animații unice;
- replici vizuale și gesturi;
- specificații pentru Blender și Unity.

## Structură planificată per erou

- `Armor_Breakdown.md`
- `Weapons_and_Props.md`
- `Ability_Icons.md`
- `Portraits.md`
- `Animation_List.md`

# 09 — Animation

## Scop

Definirea animațiilor necesare și a standardelor comune pentru unități, eroi, clădiri și elemente de mediu.

## Categorii planificate

- `Units`
- `Heroes`
- `Buildings`
- `Creatures_and_NPCs`
- `Environment`
- `UI_Animation`

## Set minim pentru unități

- idle;
- walk;
- run, dacă este necesar;
- attack;
- hit reaction;
- death;
- cast sau ability;
- victory sau acknowledgement;
- construcție, muncă ori colectare pentru unitățile specifice.

## Standarde de stabilit

- rig comun și compatibilitate între unități;
- root motion versus mișcare controlată din cod;
- frame rate și durată;
- tranziții Animator;
- direcții și variante de atac;
- evenimente de animație;
- sincronizarea cu damage, VFX și sunet;
- naming și export FBX.

## Prima țintă

Setul complet minim pentru unitatea melee folosită în vertical slice.

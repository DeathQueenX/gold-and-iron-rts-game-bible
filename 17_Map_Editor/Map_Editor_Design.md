# Map Editor Design

## Vision
The game should ship with a fully featured editor inspired by Warcraft III World Editor, modernized for large RTS projects.

## Core features
- Terrain sculpting
- Biome painting
- Road and river tools
- Resource node placement
- Trigger editor
- Cinematic editor
- AI path visualization
- Weather system
- Lighting presets
- Campaign packaging

## Object editing
Editable:
- Units
- Heroes
- Buildings
- Abilities
- Buffs
- Upgrades
- Resources
- Neutral camps
- Bosses

## Scripting
Support visual triggers and optional scripting API for advanced campaign logic.

## Debug overlays
- Pathfinding
- Vision
- Resource heatmap
- AI regions
- Spawn points
- Capture radii

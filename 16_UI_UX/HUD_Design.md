# HUD Design

## Core principles
- Maximum readability.
- Warcraft III / StarCraft II style information density.
- Minimal clicks.
- Full keyboard support.

## Layout
Top bar: resources, population, timers.
Bottom left: minimap.
Bottom center: unit portrait and abilities.
Bottom right: command card.

## Requirements
- Color-blind modes.
- UI scaling.
- Hotkey overlays.
- Hero cooldown visualization.
- Objective tracker.

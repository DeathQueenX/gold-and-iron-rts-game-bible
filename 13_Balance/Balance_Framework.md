# Balance Framework

## Purpose

Acest document definește regulile de balans pentru unități, eroi, economie și hărți.

## Core principle

Facțiunile sunt asimetrice, dar fiecare trebuie să aibă:

- un early game viabil;
- o cale de revenire după pierderi;
- counters clare;
- cel puțin două strategii principale;
- vulnerabilități exploatabile.

## Counter triangle draft

### Infantry
- Light infantry beats workers/casters.
- Heavy infantry beats light infantry.
- Anti-armor magic beats heavy infantry.
- Siege and cavalry punish ranged clumps.

### Cavalry / mobility
- Cavalry beats ranged and exposed siege.
- Pikes beat cavalry.
- Terrain can reduce cavalry efficiency.

### Siege
- Siege beats buildings and static defenses.
- Fast units beat unprotected siege.
- Magic shields can mitigate siege but not ignore it.

### Magic
- Solar beats corruption and light armor.
- Iron beats physical pressure and displacement.
- Water beats formations and fire/steam positioning.
- Abyssal beats static heavy defenses but is countered by purification.

## Hero balance rules

- Heroes should not replace armies.
- Heroes should create tactical windows.
- Each ultimate must be counterplayable.
- Hero death should matter but not instantly decide the game.
- Support heroes should scale through army value, not raw damage.

## Economy balance rules

- First expansion should be contestable but not mandatory before minute 3.
- Resource nodes should create conflict lanes.
- Faction-specific resources should be strong but not hard-lock unit production.
- Trade lanes must be vulnerable to harassment.

## Map balance rules

- Every map needs alternate routes.
- Chokepoints must have counters or flanking paths.
- Water maps must include land win conditions.
- Dry maps must give Thalassia non-naval options.
- Ashlands hazards must reward planning, not random punishment.

## Testing checklist

- Can each faction survive early rush?
- Can each faction break turtling?
- Does each faction have at least one comeback path?
- Are hero ultimates too decisive?
- Are naval units useful without deciding every map?
- Does corruption create choices rather than annoyance?

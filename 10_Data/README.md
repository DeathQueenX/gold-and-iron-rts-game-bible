# Legacy Prototype Data

Fișierele din acest folder sunt prototipuri timpurii și pot conține valori mai vechi sau seturi de date mai largi decât baza actuală.

## Regula de utilizare

- `Database/` este sursa canonică pentru valori de gameplay și import în Unity.
- `10_Data/` rămâne temporar pentru referință și migrare.
- Dacă aceeași entitate apare în ambele locuri, valorile din `Database/` au prioritate.
- Fișierele din acest folder nu trebuie importate direct în build-ul jocului.

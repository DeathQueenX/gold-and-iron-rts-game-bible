# Map Editor Overview

## Core tools
- Terrain painting
- Height editing
- Water depth
- Road and bridge placement
- Resource nodes
- Starting locations
- Objectives and capture zones
- Creep camps
- Trigger scripting

## Validation
The editor should detect:
- unreachable starting locations;
- missing resources;
- invalid naval paths;
- blocked objective zones;
- absent victory conditions.

## Export
Maps package terrain, object placement, scripts, preview image, metadata, and localization keys.

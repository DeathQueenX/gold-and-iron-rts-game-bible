# Gold and Iron RTS Game Bible

Repository pentru documentația de design a unui joc RTS inspirat din universul **Gold and Iron**.

Obiectivul proiectului este construirea unei documentații folosibile de game designers, level designers, artiști, programatori și sound designers.

## Structură inițială

- `00_Project/` — viziune, piloni, scop, buclă de joc.
- `01_World_Bible/` — istorie, cronologie, geografie, magie, resurse.
- `02_Factions/` — facțiuni principale și relații diplomatice.
- `03_Heroes/` — eroi, roluri RTS, abilități și voice lines.
- `04_Units/` — unități militare, navale, magice și neutre.
- `05_Buildings/` — clădiri, upgrade-uri și tech unlocks.
- `06_Bestiary/` — creaturi, boss-uri și creep camps.
- `07_Gameplay/` — economie, control mapă, AI, combat, victory conditions.
- `08_Campaign/` — campanie, misiuni, cinematics.
- `09_Art_Audio/` — art direction, UI, VFX, audio, voice.
- `10_Data/` — CSV/JSON pentru prototipare.

## Reguli de lucru

1. Nu se copiază text din romane.
2. Documentația adaptează și extinde elementele lumii pentru un RTS.
3. Fiecare folder trebuie să conțină fișiere reale, nu directoare goale.
4. Datele CSV/JSON trebuie să fie suficient de curate pentru import într-un prototip.
5. Fiecare iterație va extinde repository-ul prin commit-uri tematice.

## Status

Versiune inițială: `v0.1-foundation`.

Această versiune creează fundația proiectului: lore, facțiuni, unități de bază, eroi principali și sisteme RTS principale.

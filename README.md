# Gold and Iron RTS Game Bible

Bază de design și date pentru un RTS fantasy realizat în Unity.

## Conținut

- `00_Project`–`24_Map_Editor`: lore, facțiuni, eroi, unități, clădiri, gameplay, campanie, artă, AI, balans și sisteme tehnice.
- `Database/`: sursa principală de date pentru unități, eroi, clădiri, abilități, resurse, tehnologii, upgrade-uri, AI, damage, armor, formații și veterancy.

## Facțiuni

- Ferrum — industrie, armură grea și asediu.
- Aurelia — magie solară, disciplină și formații.
- Thalassia — comerț, mobilitate și război naval.
- Ashlands — corupție, creaturi și presiune PvE.

## Status

Fundația de design este creată. Prioritatea actuală este extinderea fișierelor din `Database/` pentru import și prototipare în Unity.
